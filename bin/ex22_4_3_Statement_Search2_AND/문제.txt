 조건
   1) 여러개의 조건이 동시에 만족하는 쿼리를 실행보는 프로그램을 작성해 봅니다.
   2)  처음에 9번을 선택하면 emp테이블의 모든 내용이 출력되고 종료됩니다.
      
   3)  검색 조건이 한 개인 경우 다음에 조회할 조건에 9번을 선택하면 검색 조건이 완료됩니다.
       예)  조회할 조건을 선택하세요>1
             조회할 사원번호를(을) 입력하세요>7788
             조회할 조건을 선택하세요>9 
       실행쿼리문) select * from emp where empno= 7788 

   4)  검색 조건이 두 개인 경우 
        예) 조회할 조건을 선택하세요>3
             조회할 직급를(을) 입력하세요>MANAGER
             조회할 조건을 선택하세요>5
             조회할 입사일자를(을) 입력하세요>1981-06-09
             조회할 조건을 선택하세요>9
        실행 쿼리문)  select * from emp where job= 'MANAGER' and hiredate= '1981-06-09' 
  
   5)main() 일부분
      main()내의 일부 소스 입니다. 
      public static void main(String[] args) {
		String[] search_word = menuselect();
		search(search_word);
	}
